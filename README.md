 ## AI Interview Coach
An AI-powered web application to help users practice interview questions and get instant feedback.

## Features
- ✅ Select common interview questions
- ✅ Type your answers
- ✅ Get AI feedback
- ✅ Scores for:
  - Confidence
  - Clarity
  - Grammar
- ✅ Clean UI
- ✅ Works on mobile & laptop
- ✅ Deployed online

## Tech Stack
- Python
- Flask
- Groq API (LLM)
- HTML / CSS
- Render (Deployment)
